# Data 合规星 · 产品简报（脱敏版）

> 面向面试官 / 投资人一页纸。**不含**任何登录账号、密码或内部环境地址。完整叙述见团队《产品手册》。

---

## 一句话

**Data 合规星** 是基于 **腾讯云智能体开发平台（ADP）** 的法律科技产品，定位为移动应用 **隐私政策智能审查官**：覆盖 **隐私政策审查与修订**、**个人信息保护合规评估报告**、**数据合规法律咨询**，把法规与实务经验沉淀为可执行的智能体工作流。

---

## 行业痛点（摘要）

监管对 APP 隐私政策从「形式合规」转向「实质审查」；人工审查 **门槛高、成本高、风险高、交付慢**。通用大模型易 **幻觉**；纯技术检测工具难产出 **可落地的法律文本**。本产品打通 **「发现问题 → 修订文书 → 可追溯交付」**。

---

## 产品理念 D.A.T.A.

| 字母 | 含义 |
|------|------|
| **D** | Deep — 行业与监管语境下的深度理解 |
| **A** | Accurate — 风险点与条款级精准识别 |
| **T** | Trustworthy — 结论可追溯至法规与实务口径 |
| **A** | Actionable — 输出可编辑的 Word/Markdown 交付物 |

---

## 功能矩阵：一体两翼

- **一体**：隐私政策 **智能审查与修订**（解析 → 规则比对 → 模板/范本驱动修订 → **修订痕迹** → 对话式打磨）。  
- **两翼之一**：**个人信息保护合规评估报告**（隐私政策承诺 + **15 道问卷** 双重采集，高/中/低风险定级，结构化报告）。  
- **两翼之二**：**数据合规法律咨询**（权威知识源 + **RAG**，多轮对话，贴近监管实务）。

---

## 目标用户

1. **中小企业 / 独立开发者** — 低边际成本的「虚拟合规官」。  
2. **企业法务** — 标准化初筛与初稿，释放精力做复杂判断。  
3. **律所初级与实习生** — 对齐 **39 类** APP 场景范本与审查清单，提升交付一致性。

---

## 技术架构（四维摘要）

| 层级 | 要点 |
|------|------|
| **知识层** | 法规与国家标准库、**39 类**行业隐私政策范本、行政执法与案例素材、两大评估体系（隐私政策 **9×32** 指标；个保评估 **3×19** 指标）。 |
| **模型层** | **Temperature 0.2**、**Top-P 0.3** 抑制发散；结构化提示词；Python 代码节点处理文档与交付 URL。 |
| **应用层** | 典型违规表述替换、缺失章节补全、文本与问卷交叉校验。 |
| **交互层** | 人机协同改稿；交付物保留 **删除线 / 加粗 / 标红** 等法务友好痕迹；评估结论 **有据溯源**。 |
